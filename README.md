# Multimodal-Emotion-Recognition
Combine FER (Facial Emotion Recognition), SER (Speech Emotion Recognition) and NLP (Natural Language Processing) using a late fusion approach.
 \
 1. clone the repo locally
 2. cd into it, and inside a new virtual env (important!) , do: 'pip install -r requirements.txt' to install dependencies.
 3. launch the download.py script from inside /data to install the datasets locally. They will appear in the /data/raw folder
